Arbre Components

caution

These components works inside rb or arb file within the ActiveAdmin context.

We've extended ActiveAdmin's components and applied Bootstrap styles. Here we will only list the additions and changes we've made. Please see ActiveAdmin for other supported Arbre components and usage guides.

Icons#

We've added a helper to render icons inside ActiveAdmin. The default icons are Bootstrap Icons using the bi class. If you wish, you can change the default icon class like below:

config/active_admin_bootstrap/.../css_classes.yml
---icon: your-custom-class

To render the icon inside Arbre, call:

aa_icon("bi-alarm")

Panels#

See ActiveAdmin/arbre-components/panels for basic configurations and usage guides.

Optional Title#

We've removed the requirements for the title param. If you don't pass in the title param or passed in an empty value, the panel header will not be visible.

panel do  h1 "Hello world"end
panel "" do  h1 "Hello world"end

HTML Options#

You can add additional attributes to the panel's header and body using header_html and body_html.

  • Header
  • Body
panel header_html: {class: "my-header-class"} do  h1 "Hello world"end
panel body_html: {class: "my-body-class"} do  h1 "Hello world"end

Columns#

We've replaced the inline css in ActiveAdmin's columns with Bootstrap/layout/grid. See ActiveAdmin/arbre-components/columns for basic configurations and usage guides.

caution

:max_width and :min_width options no longer works.

Gutter#

You can change the gutter between columns by passing in your own gutter class.

  • Ruby
  • HTML
columns(class: "g-5") do  ...end
<div class="g-5 row">  ...</div>

Sizing#

You can change the breakpoint of the columns by passing in size.

  • Ruby
  • HTML
columns do  column(class: "custom-column", size: "md", span: 8) do    ...  end
  column(class: "custom-column", size: "md", span: 4) do    ...  endend
<div class="g-3 row">  <div class="col-md-8 custom-column">    ...  </div>
  <div class="col-md-4 custom-column">    ...  </div></div>

Table For#

See ActiveAdmin/arbre-components/table_for for basic configurations and usage guides.

Scrolling#

You can add scrolling to tables by usings Bootstrap's .table-responsive class.

div class: "table-responsive" do  table_for order.payments do    ...  endend

Tabs#

We've replaced jQuery with Bootstrap. See ActiveAdmin/arbre-components/tabs for basic configurations and usage guides.

HTML Options#

You can add additional attributes to the tab's nav and content using :nav_html and :content_html.

  • Nav
  • Content
tabs nav_html: {class: "my-tabs-nav"} do  tab "tab-1" { ... }  tab "tab-2" { ... }end
tabs content_html: {class: "my-tabs-content"} do  tab "tab-1" { ... }  tab "tab-2" { ... }end

HTTP Tabs#

For load-intensive tabs, you might want the page to refresh when the user visits another tab. To achieve this, you can passing http: true to it.

tip

If you add an :id to the tab, it will automatically set as the anchor for the nav links.

tabs http: true, id: "test" do  tab "tab-1" { ... }  tab "tab-2" { ... }end
tip

The browser memorizes the tab parameter in the session and redirects users to the correct tab for form submissions!.

e.g., app/admin/users.rb
ActiveAdmin.register User do  form do |f|    panel do      tabs http: true, id: "user-form" do        tab :info do          f.inputs do            f.input :name          end        end        tab :password do          f.inputs do            f.input :password          end        end      end
      f.actions    end  endend
