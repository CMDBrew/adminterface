Installation

Requirements#

  • Ruby >= 2.7
  • Rails >= 6.0
  • ActiveAdmin ~> 2.9
  • Bootstrap ~> 5
  • Webpacker >= 5.0

Quick Start#

  1. Add the following to your application's Gemfile:

    Gemfile
    gem "adminterface"

  2. Then execute:

    $ bundle install

  3. Install ActiveAdmin (If you haven't installed it yet)

    $ rails g active_admin:install --use_webpacker

  4. Run the Install Generator

    $ rails g adminterface:install

  5. Migrate your database and start the server

    $ rails db:migrate$ rails server
