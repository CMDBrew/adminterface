Installation

Requirements#

  • Rails >= 6.0
  • Ruby >= 2.7
  • ActiveAdmin ~> 2.9
  • Webpacker >= 5.0

Quick Start#

  1. Add the following to your application's Gemfile:
Gemfile
source "https://rubygems.pkg.github.com/cmdbrew" do  gem "active_admin_bootstrap"end
  1. Then execute:
$ bundle install
  1. Install ActiveAdmin (If you haven't installed it yet)
$ rails g active_admin:install --use_webpacker
  1. Run the Install Generator
$ rails g active_admin_boostrap:install
